# Cpp Fast Config Web

Sitio oficial de producto + documentación para **Cpp Fast Config**, construido con **Next.js + Tailwind CSS**.

## Rutas implementadas

- `/`
- `/docs`
- `/docs/getting-started`
- `/docs/cli`
- `/docs/scaffolding`
- `/docs/ci`
- `/docs/license`

## How to run locally

```bash
pnpm install
pnpm dev
```

Abrir `http://localhost:3000`.

## Validación de producción

```bash
pnpm build
pnpm start
```
